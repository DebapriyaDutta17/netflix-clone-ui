# 🎬 Netflix Clone

A modern and responsive Netflix Homepage Clone built using HTML, CSS, and JavaScript.

This project recreates the latest Netflix India landing page UI, featuring a hero section, trending movies carousel, reasons-to-join cards, FAQ accordion, and footer.
